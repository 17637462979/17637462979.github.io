ModuleNotFoundError: No module named 'win32api'以及另一个奇葩错误

2019-10-13

在运行scrapy项目时,报错:
ModuleNotFoundError: No module named 'win32api'[缺少,那就装]
https://pypi.org/project/pypiwin32/#files
到这里下载对应的whl文件到scripts文件夹下安装,由于是在虚拟环境下,使用原来的pip 会安装到本身的环境中
所以激活虚拟环境安装
此时报错:
Fatal error in launcher: Unable to create process using '"d:\venv\scripts\python.exe" "D:\base_crawl\venv\Scripts\pip.exe" install pypiwin32-223-py3-none-any.whl'

换成pip3:

Fatal error in launcher: Unable to create process using '"d:\venv\scripts\python.exe" "D:\base_crawl\venv\Scripts\pip3.exe" install pypiwin32-223-py3-none-any.whl'

换成easy_install :
Fatal error in launcher: Unable to create process using '"d:\venv\scripts\python.exe" "D:\base_crawl\venv\Scripts\easy_install.exe" pypiwin32-223-py3-none-any.whl'

-------------->此时pip 装什么也装不上
一通乱搜之后的解决方案:
python -m pip install pypiwin32-223-py3-none-any.whl

成功=-=
